Continuando com os primeiros passos no sistema, agora é a hora de cadastrar as unidades de medidas dos produtos. Essa é uma etapa bem fácil e rápida, confira a lista abaixo das unidades mais usadas durante os cadastros de produtos.

<h2>Exemplo de lista de unidades de medidas</h2>

- Unidade
- cm
- ml
- mg
- g

Existem outros tipos de unidades, sinta-se livre para cadastrar de acordo com a sua necessidade. Essa unidade de medida é um campo obrigatório para preencher dentro do formulário de cadastro de produto.

Para acessar essa funcionalidade, clique em “Produtos/Serviços” e depois clique em “Unidades de medida”, conforme a imagem abaixo apresenta.

<img src="/unidades-de-medida-loopvet.webp" />

Para cadastrar uma nova unidade de medida, clique no botão verde “+NOVO” e preencha o campo. Após salvar, as unidades de medidas serão carregadas na tela de cadastro de produto. Se você não for mais usar alguma unidade é possível inativar, dessa forma não será mais listada no cadastro de produto.

<img src="/unidades-de-medida-loopvet-1.webp" />
