Tela 1.0 – Seção “Gerenciar Usuários”

<h2>1.11.1 – Usuários</h2>

Na subseção “Usuários”, estarão os usuários cadastrados para interagir com o sistema, suas respectivas opções de editar, representadas pelo ícone de lápis, onde se poderá alterar as informações do usuário, assim como ativá-lo ou desativá-lo, e excluir, representada pelo ícone de lixeira. Há também duas colunas, “Acesso ao sistema” que representa se o respectivo usuário possui acesso ao sistema e “Aceitou a clínica”, representando se o usuário cadastrado aceitou o e-mail de confirmação de novo usuário. O botão “+ Novo” é onde fará o cadastro, preenchendo as informações necessárias, do novo usuário.

1.5 – Permissões do novo usuário.

<h1>1.11.2 – Gerenciar comissões</h1>

Na subseção “Gerenciar comissões”, o usuário poderá associar os grupos de comissões de usuário com os grupos de comissões de produtos. Os grupos de comissões de usuário estão localizados na linha em cinza e os grupos de comissões de produtos, na primeira coluna à esquerda.

Nessa subseção, o usuário também pode criar um novo grupo de comissão de usuário e de produto. Os grupos de comissões de produtos podem ser apenas alterados nesta subseção. Para serem excluídos, a seção de “Produtos/Serviços” deve ser acessada e depois sua subseção “Grupos de Comissões”.

Tela 1.8 – Seção “Produtos/Serviços” e subseção “Cargos de Comissões”.

Na subseção “Lista de IPs”, o usuário poderá cadastrar os IPs que poderão fazer o acesso no seu sistema e gerenciar solicitações de acesso ao sistema. Isso serve, por exemplo, para limitar o local que será acessado o sistema. Ao clicar no botão “+ NOVO”, o usuário poderá adicionar um novo IP de acesso.

**Observação:** Os IPs, geralmente, são dinâmicos, ou seja, quando a rede de internet é reiniciada, seja pela queda dessa ou reiniciamento de um modem, o IP tende a mudar, portanto, essa função será **efetiva** caso o usuário tenha um IP estático (contatar sua operadora de internet para mais informações).
