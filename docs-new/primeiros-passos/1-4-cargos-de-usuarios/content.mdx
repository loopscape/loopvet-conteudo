O próximo passo é cadastrar os cargos de usuários no sistema. Ele tem a funcionalidade para nomear os usuários que têm acesso ao seu sistema. Dessa maneira, se você tem 2 funcionários com o mesmo nome, é fácil distinguir apenas pelo cargo do seu colaborador.

Geralmente são cadastros os seguintes cargos:

- Médico Veterinário
- Gerente
- Recepcionista
- Auxiliar Veterinário

Se você tem um petshop dentro da sua clínica veterinária, você pode cadastrar os seguintes cargos:

- Banhista
- Tosador

Agora o próximo passo é cadastrar todos os cargos no seu sistema, para isso acesse o menu “Gerenciar Usuários” e clique em “Gerenciar Cargos de usuário”, confira a imagem abaixo.

<img src="/gerenciar-cargos-de-usuario-loopvet-1.webp" />

Na imagem acima você pode conferir todos os cargos cadastrados na listagem e os botões para alterar e excluir um cargo. Também tem o botão “+ NOVO” para você cadastrar um novo cargo no sistema.

<img src="/gerenciar-cargos-de-usuario-loopvet-2.webp" />
